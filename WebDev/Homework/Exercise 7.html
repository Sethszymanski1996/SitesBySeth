Thinking Inside the Box and Floating & Positioning

Exercise 6 - 15pts

Readings - Chapter 14 & 15 - Learning Web Design, 5th ed.

For all Activities, students are to write their own code. If, I suspect it that students are using code from other sources, the score on the Exercise will be zero and could cause further disciplinary action under the LCC Student Code of Conduct, Article III, part B.

Activities

  • 1. Create a new HTML page using snail_mail.txt for the content. Also, create an external stylesheet. There are two images in the ex6-img.zip folder for use in the exercise.

  • 2. Make sure the content is properly formatted with both a h1 and h2. There should also be three h3 elements and at least three paragraphs for the content. Add a div element to the content area.

  • 3. Add the following to the CSS file:
     -a body property with a margin of 10px on all sides.
     -change the default font-property for the document
     -change the default color of all headings
     -the div element needs a border that looks presentable.
     -the div should also have a width of 75 or 80% and padding of at least 3px (or more)
     -paragraphs should have a margin of at least 10px to cause the content to indent
     -there should be a sidebar id that floats to the right, has the escargot.jpg as its background image and a width of 250px and a height of 345px
     -using blue-ribbon-sm.png is optional
  • NOTE: The ex6-sample image is an example of how the exercise can be completed and is for reference only.
  • 4. Create a link to this exercise from your home page and from the new page to the homepage. 

Turn In

  1. 1. Validate the document for HTML5 - the file doesn't have to be valid to be turned in, but does should use the HTML5 doctype and proper meta tag for the character type.

    2. Save the files on your computer or to a portable storage device.

  2. 3. Using Filezilla or other FTP Client, upload the files to the server prior to the posted due date. Be certain to upload the CSS file and all HTML files to the server.
